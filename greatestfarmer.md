<h3> Functionality Highlights</h3>
<ul>
<li>Login, Sign Up</li>
<li>User's dashboard - Overview of Subscriptions, Order, Saved Post and Cart</li>
<li>Farmer account gets to maintain its own inventory and write posts</li>
<li>Buyer account gets to order, leave reviews and comments, see blog posts and save to the account</li>
<li>Blog posting and Keyword Searching</li>
<li>Farmer and Farming Produce Rating System</li>
</ul>

<h3> Role within the project </h3>
<p>The project was done in a team of 3 people. I was responsible for the <b>database design </b> (coming up with and optimizing relational schemas), <b>writing back-end codes</b> (for the "Farmer" module) in PHP and <b>connecting to website's front-end </b> using Javascript (jQuery).</p>
